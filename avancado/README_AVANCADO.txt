# Pasta avançado

Esta pasta contém utilitários e documentação técnica para uso avançado.

Arquivos:
- executar_treinamento.bat: treina o modelo (uso único ou quando atualizar dados/regras)
- rules_summary.txt: resumo das regras de detecção de PII
- test_rules.py: valida as regras com casos difíceis e gera relatórios em reports/

Uso rápido:
1) Treinar:
   avancado\executar_treinamento.bat

2) Testar regras:
   python avancado\test_rules.py
